# 🤖 PM Knowledge Assistant — RAG System

A Retrieval-Augmented Generation (RAG) system
that answers questions from PM project documents.

## RAG Pipeline
Document → Chunk → Embed → Store → Retrieve → Answer → UI

## Tech Stack
- Python 3.14
- LangChain
- ChromaDB
- OpenAI GPT-4o-mini
- Streamlit

## How to run
pip install -r requirements.txt
set OPENAI_API_KEY=your_key_here
python -m streamlit run app.py
