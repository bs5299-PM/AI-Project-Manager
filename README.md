## 🚀 Featured Projects

### 🎙️ LectureLens AI — Real-Time Lecture Intelligence System (FLAGSHIP PROJECT)
A real-time AI system that converts live lecture audio into structured learning signals (Action Items + Key Concepts) using streaming speech-to-text and LLM-based extraction.

**What it solves:**
Students often struggle to simultaneously listen and take notes during fast-paced lectures, leading to missed information and incomplete understanding.

**Solution:**
LectureLens processes live audio streams, transcribes them using Whisper, and extracts structured insights using GPT-4o-mini in real time.

- 🔴 Extracts Action Items (assignments, deadlines, tasks)
- 🔵 Extracts Key Concepts (definitions, frameworks, ideas)
- ⚙️ Stack: Whisper, GPT-4o-mini, Streamlit
